# 副業アクションプラン

選択した副業アイデアについて、90日間の行動計画＋XLSXスプレッドシート＋派生ロードマップを策定する。

## 入力

選択されたアイデアと、可能であればユーザーのプロファイル。$ARGUMENTS で親スキルから渡された場合はそれを使用する。

## 90日間ロードマップ

### Phase 1: 準備期間（1-2週目）
- **環境構築**: 必要なツール・アカウントのセットアップ
- **学習計画**: 不足スキルの学習リソースと期間
- **市場リサーチ**: 競合調査、ターゲット顧客の理解
- **具体的タスク**: 日単位のTo-Doリスト

### Phase 2: MVP・初期テスト（3-4週目）
- **MVP定義**: 最小限の成果物・サービスの定義
- **テスト方法**: 小さく始めて反応を確認する方法
- **初期フィードバック**: 誰にどう見せて意見をもらうか

### Phase 3: 初収益（5-8週目）
- **価格設定**: 初期価格の決め方
- **集客方法**: 最初の顧客をどう見つけるか
- **営業活動**: 具体的なアプローチ方法

### Phase 4: 改善・拡大（9-12週目）
- **データ分析**: 何を測定し、どう改善するか
- **スケールアップ**: 収益を伸ばすための次のステップ
- **仕組み化**: 効率化・自動化の検討

## 必要ツール・プラットフォーム

具体的なサービス名と費用をリストアップ（無料 / 有料必須 / あると便利 の3分類）。

## KPI設定

各フェーズ終了時に確認する指標（セットアップ完了 / MVP完成 / 初収益金額 / 月収目標）。

## 法務・税務チェックリスト

- 就業規則の副業規定を確認
- 開業届の要否を判断（年間所得20万円超の見込みがあれば推奨）
- 青色申告承認申請の検討
- 経費計上可能な支出の記録開始
- インボイス登録の要否を検討（B2B取引の場合）

## 派生ビジネスロードマップ（90日後の展望）

evaluatorの派生ビジネス分析がある場合はそれを引用する:

- **Month 4-6**: Tier 1派生の準備
- **Month 7-12**: Tier 1派生の本格開始
- **Year 2以降**: Tier 2-3への展望

## XLSX スプレッドシート出力

`scripts/generate_xlsx.py` を使用してXLSXも出力する。入力JSONスキーマは [xlsx-schema.md](references/xlsx-schema.md) を参照。

### 生成手順

1. アクションプランのデータを [xlsx-schema.md](references/xlsx-schema.md) のスキーマに従ってJSON化
2. `python3 scripts/generate_xlsx.py output.xlsx input.json` を実行
3. 生成されたXLSXファイルのパスをユーザーに伝える

openpyxlが未インストールの場合はマークダウン形式のみで出力し、インストール方法を案内する。

## 挫折防止のコツ

- 最初の30日間は収益を期待しない
- 週次で振り返りの時間を設ける（日曜15分）
- 小さな成果を記録する（モチベーション維持）
- コミュニティや仲間を見つける
