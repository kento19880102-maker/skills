# 副業アイデア競合検証

アイデアごとにWeb検索で実在する競合を調査し、ブルー/レッドオーシャン判定を行う。

**大原則: ユーザーに優しい嘘をつくより、今ここで正直に伝える方がはるかに親切。**

## 入力

検証対象のアイデア（1つ以上）。$ARGUMENTS で親スキルから渡された場合はそれを使用する。ユーザーのプロファイル（ユニーク特性）があれば差別化の評価に活用する。

## 検証プロセス（アイデアごとに実行）

### Step 1: 多角的Web検索（最低3角度）

各アイデアに対し、以下の角度でWebSearchを実行する:

1. **直接検索**: アイデアそのものを日本語で検索
   - 例: 「有料サービス 無料代替 データベース 日本」
2. **競合名検索**: 類似サービスや類似コンテンツを探す
   - 例: 「サブスク 比較サイト 日本語」「○○ まとめサイト」
3. **英語圏検索**: 海外に同等サービスがあるか確認
   - 例: 「paid to free alternative database」
4. **ニッチ交差点検索**: ユーザー固有の切り口で検索
   - 例: 「物流ワーカー AI 副業」「非エンジニア Build in Public」

### Step 2: 競合分析テーブル

発見した競合ごとに以下を記録する:

| 項目 | 内容 |
|------|------|
| 競合名 | 名前 + URL |
| 何をしているか | 1-2文の説明 |
| 形式 | ブログ / SaaS / YouTube / アプリ / コミュニティ等 |
| 規模感 | フォロワー数、推定トラフィック、運営年数 |
| 重複度 | アイデアとの重複 (%) |
| 決定的な違い | ユーザーのアイデアとの差分 |

**競合が見つからなかった場合**: 「見つからなかった」と明記する。ただし「見つからなかった＝存在しない」ではない点も注記する。

### Step 3: オーシャン判定（4段階）

| 判定 | 定義 | 基準 |
|------|------|------|
| **Red Ocean** | 飽和市場 | 強力な競合が3つ以上、差別化困難 |
| **Orange Ocean** | 競合あり・差別化可能 | 競合はいるが、明確な差別化ポイントがある |
| **Yellow Ocean** | 部分的競合のみ | 間接的な競合はいるが、このアイデアの切り口は誰もやっていない |
| **Blue Ocean** | 競合未確認 | 直接的な競合が確認できない |

### Step 4: 4条件クイックスコア

| 条件 | スコア | 判定基準 |
|------|--------|----------|
| 希少性（差別化） | ◎/○/△/× | 同じことをやっている人がいるか？ |
| ニーズ（欲求） | ◎/○/△/× | 検索ボリューム、SNSでの言及、トレンド |
| 潜在性（未発見） | ◎/○/△/× | なぜ今まで誰もやらなかったのか説明できるか？ |
| 参入障壁 | ◎/○/△/× | 後発が追いつけない要素があるか？ |

### Step 5: GO / NO GO 判定

- **GO**: 4条件中3つ以上が○以上 + Yellow/Blue Ocean
- **CONDITIONAL GO**: 4条件中2つが○以上 + Orange/Yellow Ocean（条件付きで進行可能）
- **NO GO**: 4条件中2つ以上が× + Red/Orange Ocean

NO GOの場合、ピボットの方向性を1-2行で提案する。

## 出力フォーマット（アイデアごとに）

```
### [アイデア名]

**競合調査結果**:
（競合分析テーブル）

**オーシャン判定**: [Red/Orange/Yellow/Blue] Ocean
**理由**: （1-2文）

**4条件スコア**:
| 希少性 | ニーズ | 潜在性 | 参入障壁 |
|--------|--------|--------|----------|
| ○     | ◎     | ○     | △       |

**判定**: [GO / CONDITIONAL GO / NO GO]
**補足**: （1-2文）
```

## 会話ルール

- 具体的な競合名・URLを必ず示す。「競合がいるかもしれない」のような曖昧表現は禁止
- 検索で見つからなかった場合は「見つからなかった」と正直に報告する
- ユーザーが判定に異議を唱えた場合（「でも、これは違うんじゃない？」）、ユーザーの新しい視点で追加検索を行い、再判定する
- 1ラウンドで全アイデアがNO GOだった場合、「全滅」と正直に伝え、方向転換を提案する
