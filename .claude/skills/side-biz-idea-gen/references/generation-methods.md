## サービス支出分析法

ユーザーが日常的に使っているサービス（有料・無料）から副業アイデアを逆算する。

### パターン1: 有料→無料（節約パッケージ型）

ユーザーが払っている有料サービスの無料代替を発見・検証し、その知識をコンテンツ化またはDB化する。

**思考プロセス**:
1. ユーザーの有料サービスを一覧化
2. 各サービスの無料代替を探す
3. 実際に乗り換えてみる（n=1実験）
4. 比較結果をコンテンツ/DBとして公開

**例**:
- dラボ（月額○円）→ YouTube無料チャンネル、Podcast → 比較記事
- 有料天気アプリ → 無料天気アプリ → 比較DB
- 有料家計簿 → 無料家計簿 → 乗り換えガイド

### パターン2: 無料→収益化（価値発見型）

ユーザーが無料で使っているサービスの中から、収益化のヒントを見つける。

**思考プロセス**:
1. ユーザーの無料サービス利用パターンを分析
2. 「なぜ無料なのに使い続けるのか」の価値を特定
3. その価値を別の形で提供するビジネスを考える

**例**:
- YouTube無料視聴 → 特定ジャンルのキュレーション有料メルマガ
- Google検索 → 特定分野の検索結果を構造化したDB
- 無料メモアプリ → 特化型テンプレート販売

### パターン3: 不満→解決（ペインポイント型）

ユーザー自身や周囲の人の不満・困りごとからビジネスを発想する。

**思考プロセス**:
1. 不満・困りごとのリストを分析
2. 「同じ不満を持つ人はどれくらいいるか？」を推定
3. 解決策をサービス化できるか検討

## ユニーク交差点法

ユーザーの複数の属性を掛け算し、「この組み合わせは自分だけ」というニッチを発見する。

### 交差点の作り方

1. ユーザーの属性を3-5個リストアップ
2. 2-3個の属性を掛け合わせる
3. その交差点にいる人が「他にいるか？」を考える
4. いなければ、その交差点が独自のポジション

### 属性カテゴリ

- **職業**: 物流、営業、事務、エンジニア、医療、教育...
- **ツール**: Claude Pro、Notion、Excel、特定のアプリ...
- **知識源**: dラボ、特定のYouTuber、学術論文、業界誌...
- **ライフステージ**: 子育て中、独身、介護中、転職後...
- **趣味・関心**: 科学、筋トレ、料理、読書、ゲーム...

### 例: 交差点からのアイデア生成

| 交差点 | ニッチ | アイデア |
|--------|--------|----------|
| 物流ワーカー × AI | 非エンジニアのAI活用 | AI活用記録コンテンツ |
| 子育て × 科学 | エビデンスベース育児 | n=1育児実験記録 |
| 現場労働 × サブスク | ブルーカラーの節約 | 現場ワーカー向け代替DB |
| 介護 × テクノロジー | 介護テック | 介護×IT活用ガイド |

## 4条件スコアリング基準

### 希少性（差別化・唯一無二）

| スコア | 基準 |
|--------|------|
| ◎ | 日本語圏で同じことをやっている人が確認できない |
| ○ | やっている人はいるが、切り口・ターゲットが明確に異なる |
| △ | 類似のコンテンツ・サービスが複数あるが、差別化の余地あり |
| × | レッドオーシャン。多数の競合が存在し、差別化困難 |

### ニーズ（欲求）

| スコア | 基準 |
|--------|------|
| ◎ | 社会トレンドに合致 + 検索ボリュームが高い + SNSで頻繁に言及 |
| ○ | 一定の需要がある。検索されている |
| △ | ニッチな需要。特定の層には刺さるが市場は小さい |
| × | 需要が確認できない。誰も求めていない可能性 |

### 潜在性（他が気づいていない）

| スコア | 基準 |
|--------|------|
| ◎ | 需要はあるのに誰もやっていない。明確なギャップ |
| ○ | 部分的にカバーされているが、このアプローチは新しい |
| △ | 既に気づいている人がいて、参入が始まっている |
| × | 既に多くの人が気づいており、市場が形成済み |

### 参入障壁（他が中々入ってこれない）

| スコア | 基準 |
|--------|------|
| ◎ | 構造的なモート（データ蓄積、ネットワーク効果、独自の経験） |
| ○ | 時間経過で強くなるモート（先行者優位、継続蓄積） |
| △ | 弱いモート。他の人も始められるが、続ける人は少ない |
| × | モートなし。誰でもすぐ始められ、追いつける |
