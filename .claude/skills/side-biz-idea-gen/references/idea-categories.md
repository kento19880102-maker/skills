## デジタルプロダクト

**ビジネスモデル**: 一度作成した成果物を繰り返し販売。限界費用がほぼゼロ。

**主な商品例**:
- 電子書籍（Kindle、note）
- デザインテンプレート（Canvaテンプレート、Figmaテンプレート）
- 写真・イラスト素材（PIXTA、Adobe Stock、Shutterstock）
- Notion / スプレッドシートテンプレート
- プログラミング教材・ソースコード

**主要プラットフォーム**:
- Amazon Kindle Direct Publishing（電子書籍）
- note（記事・マガジン販売）
- BOOTH（同人・クリエイター向け）
- Gumroad / lemonsqueezy（グローバル向けデジタル販売）
- PIXTA / Adobe Stock（ストック素材）

**収益レンジ**: 月0〜5万円（初期）→ 月5〜50万円（軌道に乗った場合）

**スキルレベル**: 初級〜中級（専門性が高いほど単価が上がる）

**初期コスト**: 0〜5万円（ツール費用程度）

**特徴**:
- ストック型収入で時間の切り売りにならない
- 初期は収益ゼロの期間がある
- 品質とマーケティングの両方が必要

## フリーランス・受託

**ビジネスモデル**: 自分のスキルを時間単位またはプロジェクト単位で提供。

**主な業種**:
- Web制作（コーディング、WordPress構築）
- デザイン（ロゴ、バナー、UI/UX）
- ライティング（SEO記事、コピーライティング、取材記事）
- 翻訳（英日、中日、技術翻訳）
- 動画編集（YouTube動画、企業VP）
- プログラミング（アプリ開発、データ分析、自動化）

**主要プラットフォーム**:
- CrowdWorks（国内最大級のクラウドソーシング）
- Lancers（CrowdWorksと並ぶ大手）
- ココナラ（スキルマーケット、低単価案件も多い）
- MENTA（メンタリング特化）
- Upwork / Fiverr（グローバル案件、高単価の可能性）
- Wantedly / YOUTRUST（業務委託案件）

**収益レンジ**: 月1〜10万円（初期）→ 月10〜100万円以上（実績構築後）

**スキルレベル**: 初級〜上級（案件レベルに応じて選択可能）

**初期コスト**: 0〜3万円（PCとネット環境があれば可）

**特徴**:
- 即収益化しやすい（最短で数日〜1週間）
- 時間の切り売りになりやすいのが課題
- 実績とポートフォリオが受注に直結
- 本業のスキルをそのまま活かせることが多い

## コンテンツ制作

**ビジネスモデル**: コンテンツを通じた広告収入、アフィリエイト、スポンサーシップ。

**主な形態**:
- ブログ・Webメディア（SEOアフィリエイト、Google AdSense）
- YouTube（広告収入、スーパーチャット、メンバーシップ）
- ポッドキャスト（スポンサーシップ、有料会員）
- SNS（Instagram、X/Twitter、TikTok → 企業案件、PR投稿）
- ニュースレター（有料メルマガ、Substack）

**主要プラットフォーム**:
- WordPress + レンタルサーバー（ブログ）
- YouTube（動画）
- Spotify for Podcasters（ポッドキャスト）
- note（有料記事）
- Substack（ニュースレター）
- Instagram / X / TikTok（SNS）

**収益レンジ**: 月0円（初期数ヶ月）→ 月1〜100万円以上（フォロワー/PV次第）

**スキルレベル**: 初級から始められる（成功には継続力と差別化が必要）

**初期コスト**: 0〜3万円（サーバー代、ドメイン代程度。動画の場合は機材費追加）

**特徴**:
- 収益化まで時間がかかる（3ヶ月〜1年以上）
- ストック型資産になりうる
- 個人ブランドの構築につながる
- YouTubeの収益化条件: チャンネル登録者1,000人＋総再生時間4,000時間

## EC・物販

**ビジネスモデル**: 商品の仕入れ・製作・販売で差益を得る。

**主な形態**:
- せどり・転売（中古品、Amazon FBA）
- ハンドメイド販売（アクセサリー、雑貨）
- ドロップシッピング（在庫を持たない物販）
- OEM商品（オリジナル商品の企画・製造委託）
- 輸入販売（海外商品の国内販売）

**主要プラットフォーム**:
- メルカリ（個人間売買、最も手軽）
- Amazon（FBAで物流を委託可能）
- BASE / STORES（自社ECサイト、無料から）
- Shopify（本格的なECサイト構築）
- minne / Creema（ハンドメイド特化）
- ヤフオク!（オークション形式）

**収益レンジ**: 月1〜10万円（せどり初期）→ 月10〜100万円以上（OEM・自社ブランド）

**スキルレベル**: 初級（せどり）〜 上級（OEM・ブランド構築）

**初期コスト**: 1〜30万円（仕入れ資金が必要）

**特徴**:
- 仕入れ資金が必要（リスクあり）
- せどりは即金性が高い
- 在庫管理・発送の手間がある
- OEM/ブランド構築は高収益だが参入障壁も高い

## コンサルティング・コーチング

**ビジネスモデル**: 専門知識や経験をベースにしたアドバイス・指導をサービスとして提供。

**主な形態**:
- 経営コンサルティング（中小企業向け）
- IT・DXコンサルティング
- キャリアコーチング
- 健康・フィットネスコーチング
- 資格取得支援

**主要プラットフォーム**:
- ココナラ（相談サービス）
- ストアカ（対面/オンライン講座・コーチング）
- MENTA（メンタリング）
- ビザスク（スポットコンサル）
- タイムチケット（時間売買）
- 自社Webサイト + Calendly等の予約ツール

**収益レンジ**: 月5〜30万円（個人向け）→ 月30〜200万円以上（法人向け）

**スキルレベル**: 中級〜上級（実務経験5年以上推奨）

**初期コスト**: 0〜5万円

**特徴**:
- 高単価が狙える（時給換算で最も効率的な場合が多い）
- 本業での実績・専門性が直接的な価値になる
- 顧客獲得が最大の課題
- 時間の切り売りだが、プログラム化・パッケージ化で拡張可能

## SaaS・アプリ開発

**ビジネスモデル**: ソフトウェアを開発し、サブスクリプションまたは買い切りで販売。

**主な形態**:
- Webアプリ（B2B SaaS、B2C ツール）
- モバイルアプリ（iOS / Android）
- Chrome拡張機能 / ブラウザツール
- API / 開発者向けツール
- ノーコード/ローコードツールでのMVP構築

**主要プラットフォーム**:
- Vercel / Netlify / AWS（ホスティング）
- Apple App Store / Google Play（モバイルアプリ）
- Chrome Web Store（拡張機能）
- Product Hunt（ローンチ）
- Indie Hackers（コミュニティ）

**収益レンジ**: 月0円（開発期間中）→ 月5〜500万円以上（プロダクト次第）

**スキルレベル**: 上級（プログラミング必須、ノーコードなら中級から）

**初期コスト**: 0〜10万円（サーバー費用、ドメイン等）

**特徴**:
- 最もスケーラブルなカテゴリ
- 開発期間が長い（MVP作成に1〜3ヶ月）
- 技術力だけでなくマーケティングが成功の鍵
- 小さく始めて検証→拡大のアプローチが重要
- AI活用で開発効率が大幅に向上している

## 教育・スキルシェア

**ビジネスモデル**: 知識やスキルを教材化・サービス化して提供。

**主な形態**:
- オンライン講座（動画コース）
- ライブ講座・ワークショップ
- メンタリング・個別指導
- 教材・テキスト販売
- コミュニティ運営（有料サロン）

**主要プラットフォーム**:
- Udemy（グローバル、日本語コースも可）
- ストアカ（国内最大級のスキルシェア）
- Schoo（オンライン学習）
- MENTA（メンタリング特化）
- note / Brain（有料記事・教材）
- Discord / Slack（コミュニティ運営）

**収益レンジ**: 月1〜10万円（初期）→ 月10〜100万円以上（人気講師）

**スキルレベル**: 中級（教えられるレベルの専門性が必要）

**初期コスト**: 0〜5万円（撮影機材、マイク等）

**特徴**:
- コンテンツを一度作れば繰り返し販売できる（ストック型）
- 「教えるスキル」と「専門スキル」の両方が必要
- 受講者の成果が口コミにつながる好循環
- ライブ × 動画 × コミュニティの組み合わせが有効

## 投資・資産運用

**ビジネスモデル**: 資金を運用して利益を得る。

**主な形態**:
- 株式投資（個別株、ETF、投資信託）
- 不動産投資（ワンルームマンション、REIT）
- FX・暗号資産
- ソーシャルレンディング
- エンジェル投資

**注意**: 具体的な銘柄推奨や投資判断は行わない。一般的な情報提供のみ。

**主要プラットフォーム**:
- SBI証券 / 楽天証券 / マネックス証券（株式・投信）
- NISA / iDeCo（税制優遇制度）
- COZUCHI / OwnersBook（不動産クラウドファンディング）

**収益レンジ**: 年利3〜7%が一般的な期待値（インデックス投資）

**スキルレベル**: 初級（インデックス投資）〜 上級（個別株・不動産）

**初期コスト**: 1万円〜（少額投資可能。不動産は数十万〜数百万円）

**特徴**:
- 完全な不労所得に最も近い
- 元本割れリスクあり
- 副業禁止の会社でも投資は通常OK
- 時間はほぼ不要（インデックスの場合）
- 短期売買は副業というより投機に近い

## 日本固有の法務・税務知識

### 確定申告

- **20万円ルール**: 給与所得者で副業の所得（収入-経費）が年間20万円以下の場合、所得税の確定申告は不要（ただし住民税の申告は必要）
- **副業所得が20万円超**: 確定申告が必要（毎年2月16日〜3月15日）
- **青色申告**: 開業届＋青色申告承認申請で最大65万円の控除。帳簿付けが必要
- **白色申告**: 申請不要だが控除なし

### 開業届

- 継続的に収入を得る場合は提出推奨
- 開業日から1ヶ月以内に税務署に提出
- 無料で提出可能（freee開業 等のサービスで簡単に作成）
- 青色申告をする場合は同時に「青色申告承認申請書」も提出

### インボイス制度

- 2023年10月開始
- 課税売上1,000万円以下の免税事業者は登録任意
- B2B取引がメインの場合、取引先の要望で登録が必要になることがある
- B2C取引（個人向け販売）ならインボイス登録不要のケースが多い

### 副業と会社の関係

- 就業規則の確認が最優先
- 「副業解禁」の流れが進んでいるが、届出制の企業も多い
- 住民税の納付方法を「普通徴収」にすると会社に副業収入が通知されにくい
- 競業避止義務に注意（同業種での副業は制限されることがある）

### 経費として認められやすいもの

- PC・周辺機器（副業使用分を按分）
- 通信費（インターネット、電話の副業使用分）
- 書籍・教材・セミナー参加費
- ソフトウェア・サブスクリプション費用
- 交通費
- 家賃（自宅で作業する場合の按分）
